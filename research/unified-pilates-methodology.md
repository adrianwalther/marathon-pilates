# Unified Pilates Methodology
## Foundation for the Marathon Pilates AI Class Generator

Synthesized from three research bodies: Balanced Body University (BBU), BASI (Body Arts and Science International), and Polestar Pilates. Where methods agree, the principle is treated as established. Where they differ, the most conservative or evidence-based position is adopted.

---

## 1. Universal Movement Principles

These principles are present in all three methodologies. Every AI-generated class and every exercise cue must reflect them.

| Principle | Definition | Source Consensus |
|---|---|---|
| **Breath** | Breath leads and organizes movement. Exhale on effort/exertion. Inhale to prepare or lengthen. Never hold breath. | BBU, BASI, Polestar |
| **Axial Elongation** | The spine moves from its longest, optimal position — not compressed, not hyperextended. Preferred over "neutral spine" for its active quality. | Polestar primary; BASI + BBU aligned |
| **Core Control** | Deep stabilizers (transversus abdominis, multifidus, pelvic floor, diaphragm) activate before and during movement. Not a grip — a gentle, sustained engagement. | All three |
| **Concentration** | Full mental attention to movement quality. Not just position — sensation, quality, integration. | All three (classical origin) |
| **Control** | Precision over momentum. Movement is intentional. Compensatory patterns signal the need to modify. | All three |
| **Flowing Movement** | Transitions are part of the exercise. Movement is continuous, not segmented by pauses. | All three |
| **Precision** | Exactness of position and execution. Quality over quantity. | All three |
| **Centering** | Movement originates from the core/powerhouse and radiates outward. | All three |
| **Spinal Articulation** | The spine moves segmentally. Each vertebra participates. "Hinging" at one segment is a dysfunction, not a technique. | BASI, Polestar strong; BBU aligned |
| **Balance** | Every class must balance anterior and posterior chain work. Every spinal flexion series must be followed by spinal extension work. | BASI (structural), Polestar (principle), BBU (implied) |

## 2. Unified Class Architecture

### The Hybrid Framework
Uses BASI's Block System as the structural skeleton (most AI-encodable), informed by Polestar's developmental progression (supine → standing) and BBU's exercise vocabulary.

### 55-Minute Class Structure

| Block | Name | Duration | Function |
|---|---|---|---|
| 1 | Warm-Up | 8–10 min | Breath, body scan, spinal mobility, core connection |
| 2 | Abdominal Work | 8–10 min | Anterior chain loading, progressive lever length |
| 3 | Hip Work | 6–8 min | Hip extension, abduction, rotators; posterior chain activation |
| 4 | Spinal Articulation | 5–7 min | Segmental mobility, active recovery, spinal reset |
| 5 | Stretching | 4–5 min | Release tight muscles before next loading block |
| 6 | Full Body Integration | 6–8 min | Peak demand — full kinetic chain coordination |
| 7 | Lateral Flexion / Rotation | 4–5 min | Lateral chain and rotation — often undertrained |
| 8 | Back Extension | 5–7 min | Posterior chain, spinal extensors — counterbalance |
| 9 | Cool-Down | 3–5 min | Return to breath, release, integration, close |

**Immutable rules:**
- Block 1 (Warm-Up) is ALWAYS first
- Block 9 (Cool-Down) is ALWAYS last
- All 9 blocks must appear in a 55-minute class
- Blocks 2–8 may reorder based on class focus, but balance must be maintained

### 30-Minute Express Structure

| Block | Duration | Notes |
|---|---|---|
| Warm-Up | 4–5 min | |
| Abdominals | 5–6 min | |
| Hip Work | 4–5 min | |
| Spinal Articulation + Stretch (combined) | 4–5 min | |
| Full Body Integration | 4–5 min | |
| Back Extension | 3–4 min | |
| Cool-Down | 2–3 min | |

**30-min rule (Polestar):** Choose ONE focus. Do not attempt to cover everything. A 30-min class is a complete workout around one movement theme.

## 3. Exercise Difficulty Levels

Three-level system. Competency, not time, determines progression.

### Level 1 — Foundation
- Short levers, bilateral, slow tempo
- Spine supported or in neutral
- Breath and core established before any limb movement
- No inverted positions, no rolling, no dynamic momentum
- Population: beginners, post-rehab, older adults, prenatal (modified), first-timers

**Progression criteria (Polestar-informed):** Do not advance until:
- Breath integrates under load (no holding)
- Axial elongation maintained throughout
- No compensatory patterns (neck tension, rib flare, lumbar hinge)
- Core activation consistent without gripping

### Level 2 — Intermediate
- Longer levers, rotational elements, combined patterns
- Rolling work introduced
- Teaser I, Side Bend, Open Leg Rocker
- Population: 3+ months consistent practice, no acute contraindications

### Level 3 — Advanced
- Full classical repertoire
- Inverted positions (Roll Over, Jackknife)
- Dynamic momentum work (Swan Dive, Rocking)
- Boomerang, Control Balance, Corkscrew (full)
- Population: 12+ months consistent practice; full contraindication screening required

## 4. Sequencing Logic

### Developmental Progression (Polestar)
Supine → Sidelying → Prone → Quadruped → Seated → Kneeling → Standing

Classes should generally follow this spatial progression — it mirrors developmental movement patterns and prepares the body systematically.

### Lever Progression (BASI)
Within any loading block, sequence from shorter to longer lever:
- Knees bent before legs extended
- Arms at sides before arms overhead
- Bilateral before unilateral

### Anterior/Posterior Balance (all three)
Every session must contain:
- Spinal flexion work (Block 2)
- Spinal extension work (Block 8)
- Failure to include both is a methodology error

### Complexity vs. Difficulty (Polestar)
These are independent dimensions. Manage both:
- A movement can be low-load but high-complexity (standing balance)
- A movement can be high-load but low-complexity (bilateral squat)
- AI generator must track both dimensions

## 5. Unified Cueing Standards

### Language to USE
| Intent | Cue |
|---|---|
| Core engagement | "Gently draw the lower abdomen in and up" |
| Spinal length | "Find your axial elongation" / "Lengthen through the crown" |
| Breath | "Inhale to prepare, exhale and deepen" |
| Scapular stability | "Shoulder blades anchored and wide" |
| Pelvis | "Hip bones level and equal" / "Neutral pelvis — allow the natural curve" |
| Thoracic | "Soften the sternum" / "Ribs heavy" / "Knit your ribs" |
| Feet | "Energy through the heels" |
| Eccentric loading | "Resist the return" |
| Rotation | "Rotate from the thorax, not the neck" |

### Language to AVOID
| Avoid | Why |
|---|---|
| "Navel to spine" / "Suck in your abs" | Too aggressive; inhibits breathing; not anatomically accurate |
| "Flatten your back" | Eliminates natural lumbar curve; not a neutral position |
| "Tuck your pelvis" | Posterior tilt is not a default; use only when specifically intended |
| "Squeeze your glutes" | Global mobilizer dominance; inhibits deep stabilizers |
| "Keep your shoulders down" | Vague; use "anchored and wide" instead |
| "Pull your belly button in" | Too global; insufficient depth cue |

### Breath Pairing (universal)
- **Exhale** = effort / exertion / deepest point of curl / rotation peak / extension peak
- **Inhale** = preparation / lengthening / return / opening

## 6. Contraindication Framework

### Traffic Light System (Polestar)
Assess every client profile before generating a class:
- 🟢 **Green** — proceed, standard programming
- 🟡 **Yellow** — proceed with modifications; monitor; consider referral
- 🔴 **Red** — do not proceed without medical clearance; refer

### Condition-Specific Hard Rules

| Condition | Hard Blocks | Modifications |
|---|---|---|
| **Osteoporosis / Osteopenia** | No spinal flexion loading (Roll-Up, Criss-Cross, Teaser, Scissors); No rolling exercises; No rotation + flexion combined | Extension work emphasized; hip strengthening for fracture risk reduction; balance work |
| **Lumbar disc herniation** | No deep spinal flexion in acute phase; assess directional preference (extension vs flexion bias) | Extension often preferred; small range; neutral spine priority |
| **Cervical instability** | No head-loaded flexion; no inverted positions | Head-down option for all supine abdominal work |
| **Pregnancy 2nd/3rd trimester** | No supine after 16–20 weeks; no full prone; no inverted positions; no strong abdominal compression | Side-lying for most work; seated and standing emphasis; pelvic floor focus |
| **Diastasis recti** | No exercises that cause "coning" or "doming" at linea alba | Incline chest lift; diagonal patterns; breath-led engagement; test before any flexion work |
| **Hip replacement** | Follow surgical precautions (typically: no hip flexion >90°, no adduction past midline, no internal rotation) | Short lever; external rotation throughout; avoid deep hip circles |
| **Spondylolisthesis** | No aggressive spinal extension | Neutral spine emphasis; avoid hyperextension |
| **Shoulder impingement** | No Push-Up series; no full Side Bend (plank); no Leg Pull Front | Supported alternatives; scapular stability focus |
| **Glaucoma / Retinal detachment** | No inverted positions (Roll Over, Jackknife, Boomerang) | All non-inverted alternatives |
| **Uncontrolled hypertension** | No inverted positions; monitor exertion | Low-to-moderate intensity; breath continuous |

### Modification Framework (Polestar)
Modification trigger = **principle failure**, not pain alone.

If a client cannot maintain a principle, modify:
1. Reduce range of motion
2. Add support (prop, hands, surface change)
3. Change body orientation (supine instead of seated)
4. Substitute an exercise that trains the same intent at a lower demand

## 7. Pre-Class Client Screening (AI Intake)

Before generating any class, the AI must collect:

**Required:**
- Experience level (Foundation / Intermediate / Advanced)
- Class duration (30 min / 55 min)
- Class focus preference (core / flexibility / strength / balance / full-body)

**Contraindication flags (map to Traffic Light):**
- Lower back condition (disc, stenosis, general pain)
- Neck condition
- Osteoporosis / bone density concerns
- Pregnancy (and trimester)
- Recent surgery (type and timeline)
- Hip/knee condition or replacement
- Shoulder condition
- Hypertension (controlled vs. uncontrolled)
- Neurological condition (Parkinson's, MS, stroke)
- Hypermobility / EDS

**Behavioral learning inputs (from spec 02):**
- Previous class ratings
- Most-selected focus areas
- Props available
- Regularly flagged conditions

## 9. Exercise Library Structure

Every exercise in the system should carry:
- **Block assignment** (1–9)
- **Level** (Foundation / Intermediate / Advanced)
- **Primary muscles**
- **Breath pattern** (inhale/exhale per phase)
- **Setup cue** (alignment before movement)
- **Movement cue** (action)
- **Common error** (what to watch for)
- **Level down** (regression)
- **Level up** (progression)
- **Props** (optional enhancements)
- **Contraindication flags** (which conditions block this exercise)
