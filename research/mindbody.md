# Mindbody — Competitive Research
*Research date: 2026-03-09 | Knowledge base through August 2025*

---

## Pricing
- **Base:** ~$129–$599+/month depending on tier
- **Real-world:** $400–$800/month once add-ons stack (AI receptionist, branded app, API access, SMS)
- **Payment processing:** 2.75–3.5%+ per transaction (higher than Stripe)
- **Contracts:** Annual required; month-to-month available at higher rates
- **API access:** $150–$500+/month extra (previously free — major community grievance)

## Strengths
- Large consumer marketplace (millions of existing Mindbody accounts = built-in discovery)
- Mature, reliable booking engine for standard classes + appointments
- Comprehensive membership and pack management
- Multi-location support
- Broad third-party integrations (when you can afford API access)

## Core Feature Summary

### Booking
- Class calendar + appointment booking (separate UI paradigms — can confuse users)
- Waitlists with auto-promotion
- Booking windows, max participants, cancellation policy enforcement
- No native cross-service bundle checkout (sauna + class in one flow = not possible)

### Client Journey & Progress
- Attendance history, purchase history, staff notes per client
- **No milestone system, no progress tracking, no client-facing dashboard**
- Tags and segments for marketing purposes only
- This is a scheduling/payment tool — not a coaching or progression platform

### Memberships & Pricing
- Memberships, class packs, drop-ins, intro offers, gift cards
- Pause/freeze available
- Cross-service memberships (e.g., Pilates + Sauna credits) = clunky workarounds

### Multi-Service / Amenities
- Sauna/cold plunge treated as a "resource appointment" — functional but feels like a hack
- No native amenity access concept tied cleanly to membership tiers
- No single booking flow for class + amenity together

### App
- Consumer Mindbody app (cluttered marketplace, ~3.5–4.0 star ratings)
- Branded app only on highest/add-on tiers — significant extra cost
- No in-app progress tracking, no client-studio messaging

### Website Integration
- Embeddable widget (iframe) — limited styling customization
- API access for custom builds is expensive and throttled
- Widget often looks out of place on premium studio sites

### Admin Tools
- Notoriously complex UI — steep learning curve, extensive staff training required
- Strong reporting breadth but limited custom reporting
- Payroll, multi-location, POS all available but aging UI

### Communications
- Automated booking confirmations, reminders, waitlist notifications
- Email marketing module (generic templates)
- Messenger[ai] AI receptionist add-on (~$99–$149/mo)
- SMS = additional cost
- No two-way client-instructor messaging

## Key Pain Points

**From Studios:**
- $400–$800/mo real monthly cost
- Steep learning curve; significant staff training required
- Data lock-in — difficult to export or migrate away
- Forced toward Mindbody Payments processor
- API access now expensive; alienated developer community
- Support quality is consistently rated poor
- Feature development pace is slow (acquisitions > core improvements)

**From Clients:**
- Consumer app feels cluttered and generic
- Duplicate account issues are common
- No sense of the studio's individual brand
- Cancellation friction on mobile

## Opportunity Gap Table

| Mindbody Gap | What We Build Instead |
|---|---|
| No wellness amenity booking (sauna/cold plunge is a workaround) | Native amenity booking with real UX |
| Zero client progress tracking | Progress dashboard, milestones, instructor notes |
| No cross-service bundle checkout | Class + sauna in one transaction |
| Generic unbranded client app | Fully branded Marathon Pilates app |
| Dated, complex admin UI | Modern, intuitive admin tools |
| Expensive + data lock-in | Fair pricing, clean data portability |
| No in-app client↔instructor messaging | Two-way messaging |
| Widget customization very limited | Pixel-perfect Wix site integration |
| No milestone/loyalty features | Streaks, milestones, rewards |
